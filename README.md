# Portfolio Saya 1

Personal portfolio website for Nabil Khairi, built with Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Lucide React

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Edit Portfolio Content

Most personal details, skills, projects, journey items, and contact links are stored in:

```txt
data/portfolio.ts
```
